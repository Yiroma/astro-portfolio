Projet 1 · Projet de fin de cursus CDA

CarePlan — Gestion de plannings médicaux multi-rôles

Contexte

Projet de fin de cursus CDA (Wild Code School, 2025). Application de gestion de plannings pour cabinets médicaux avec trois rôles : Secrétaire, Médecin, Administrateur.

Problème

Séparer clairement les responsabilités entre rôles, garantir une accessibilité réelle (pas cosmétique), et industrialiser le delivery sur une stack microservices avec tests à plusieurs niveaux.

Solution

Architecture microservices modélisée UML, API GraphQL Apollo, tests Jest + Playwright E2E, Docker complet, CI/CD. Accessibilité dès la conception : navigation 100 % clavier, focus trap, validation NVDA, ARIA sur tous composants custom.

Résultat

Application livrée et soutenue avec succès. Pile alignée avec les standards industrie. Conformité RGAA et WCAG 2.1 effective, pas seulement déclarative.
